# EmoLang Compiler

EmoLang is a custom programming language that uses emoji operators.

## Features
- Custom language syntax
- Emoji arithmetic operators
- Variable assignment
- Direct execution

## Operators
➕ : addition  
➖ : subtraction  
✖️ : multiplication  
➗ : division  

## Example
let x = 5
let y = 10
let z = x ➕ y
print z

Output:
15
